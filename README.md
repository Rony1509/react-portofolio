# react-portofolio
## Description

A minimal React portfolio scaffolded with Vite. Lightweight, fast dev server and optimized production builds.

## Features

- React + Vite
- Fast HMR during development
- Production-ready build with minimal config

## Prerequisites

- Node.js 16+ (or latest LTS)
- npm (or yarn/pnpm)

## Installation

1. Clone the repo
    git clone <repository-url>
2. Change directory
    cd react-portofolio
3. Install dependencies
    npm install

## Development

Start the dev server (Vite with HMR):
npm run dev

Open the URL shown in the terminal (usually http://localhost:5173).

## Build & Preview

- Build for production:
  npm run build

- Preview the production build locally:
  npm run preview

## Available Scripts

- npm run dev — start Vite dev server  
- npm run build — create a production build  
- npm run preview — preview the production build locally

## Project Structure (typical)

- src/ — application source (components, pages, assets)  
- index.html — Vite entry  
- vite.config.* — Vite configuration  
- package.json — scripts & dependencies
